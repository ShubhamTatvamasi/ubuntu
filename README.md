# ubuntu

Open a Terminal window and type:
```bash
sudo visudo
```

In the bottom of the file, add the following line:
```bash
$USER ALL=(ALL) NOPASSWD: ALL
```

Install the Gnome desktop on Ubuntu server Edition enter:
```bash
sudo apt install ubuntu-desktop
```
